# Especialista-em-FBN-SOJA
Este repositório apresenta um projeto desenvolvido no **NotebookLM** focado na utilização de bioinsumos e na bioquímica da Fixação Biológica de Nitrogênio (FBN). O objetivo é detalhar como essa tecnologia sustenta a produtividade e a sustentabilidade da sojicultura brasileira.
